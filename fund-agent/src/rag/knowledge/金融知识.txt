# 金融知识汇总

## 一、股票基础知识

### 1. 股票定义
股票是股份有限公司发行的所有权凭证，代表持股人对公司资产的所有权。持有股票即成为公司股东，享有公司经营收益和重大决策的投票权。

### 2. 股票主要术语
- **A股**：在中国境内上市的以人民币计价交易的股票
- **B股**：以人民币标明面值、外币认购和交易的股票
- **主板**：大型成熟企业上市的市场，代码以600、000开头
- **创业板**：创新型企业上市的市场，代码以300开头
- **科创板**：科技创新企业上市的市场，代码以688开头
- **市盈率(PE)**：股票价格与每股收益的比率，用于评估股票估值
- **市净率(PB)**：股票价格与每股净资产的比率
- **股息率**：每股股息与股票价格的比率
- **涨停/跌停**：股票单日涨幅或跌幅达到规定上限

### 3. 股票交易规则
- 交易时间：周一至周五（节假日除外），上午9:30-11:30，下午13:00-15:00
- 涨跌幅限制：主板10%，创业板、科创板20%，ST股票5%
- 最小交易单位：100股（1手）
- T+1交易制度：当天买入股票，次日才能卖出

### 4. 股票风险
- 市场风险：整体市场下跌导致股票价格下降
- 行业风险：特定行业周期性波动
- 公司风险：企业经营状况恶化
- 流动性风险：股票难以快速变现
- 系统性风险：宏观经济因素导致的市场整体风险

## 二、基金基础知识

### 1. 基金定义
基金是一种集合投资方式，由基金管理人将众多投资者的资金集中起来，投资于股票、债券、货币市场等金融工具，以实现资产增值。

### 2. 基金分类

#### 按投资标的分类
- **股票型基金**：主要投资于股票，比例不低于80%
- **债券型基金**：主要投资于债券，比例不低于80%
- **混合型基金**：同时投资于股票和债券
- **货币型基金**：投资于短期货币市场工具
- **指数型基金**：跟踪特定指数的基金
- **QDII基金**：投资于境外市场的基金

#### 按运作方式分类
- **开放式基金**：份额可随时申购赎回
- **封闭式基金**：份额固定，可在二级市场交易
- **定期开放基金**：按固定周期开放申购赎回

#### 按募集方式分类
- **公募基金**：公开向大众募集
- **私募基金**：向特定合格投资者募集

### 3. 基金主要术语
- **基金份额**：投资者持有基金的单位数量
- **基金净值**：每份基金的价值，等于基金总资产/总份额
- **累计净值**：基金成立以来累计的净值，包括分红
- **申购**：买入基金
- **赎回**：卖出基金
- **基金定投**：定期定额投资基金的理财方式
- **基金规模**：基金管理的资金总量
- **夏普比率**：衡量基金风险调整后收益的指标

### 4. 基金费用
- **管理费**：基金管理人收取的费用，通常每年1%-2%
- **托管费**：基金托管银行收取的费用，通常每年0.1%-0.5%
- **申购费**：购买基金时收取的费用
- **赎回费**：卖出基金时收取的费用
- **销售服务费**：支付给销售机构的费用

### 5. 基金选择要点
- 了解基金的投资策略和风险等级
- 关注基金经理的管理能力和业绩稳定性
- 考察基金的历史业绩，但不能代表未来
- 关注基金规模和流动性
- 了解基金的费用结构

## 三、债券基础知识

### 1. 债券定义
债券是发行人向投资者发行的债务凭证，承诺按约定利率支付利息并到期偿还本金。债券是一种固定收益类投资工具。

### 2. 债券要素
- **面值**：债券的票面金额，通常为100元
- **票面利率**：债券年利息与面值的比率
- **到期期限**：债券从发行到偿还本金的期限
- **发行价格**：债券发行时的价格
- **偿还方式**：到期一次性还本付息或分期付息到期还本

### 3. 债券分类

#### 按发行主体分类
- **政府债券**：由政府发行的债券，如国债、地方政府债
- **金融债券**：由金融机构发行的债券
- **企业债券**：由企业发行的债券
- **公司债券**：股份有限公司发行的债券

#### 按利率形式分类
- **固定利率债券**：票面利率固定
- **浮动利率债券**：票面利率随市场利率调整
- **零息债券**：不支付利息，以折价方式发行

#### 按担保方式分类
- **信用债券**：无担保债券
- **担保债券**：有抵押或信用担保

### 4. 债券主要术语
- **久期**：债券价格对利率变化的敏感度
- **收益率**：债券投资的回报率
- **到期收益率(YTM)**：持有到期的实际收益率
- **信用评级**：对债券发行人信用风险的评级
- **债券基金**：主要投资于债券的基金

### 5. 债券风险
- **利率风险**：市场利率上升导致债券价格下跌
- **信用风险**：发行人违约无法还本付息
- **流动性风险**：债券难以在二级市场变现
- **通货膨胀风险**：通货膨胀侵蚀债券收益
- **汇率风险**：外币债券面临汇率波动

## 四、货币市场基础知识

### 1. 货币市场定义
货币市场是期限在一年以内的短期资金借贷市场，主要功能是满足短期资金需求和进行流动性管理。

### 2. 货币市场工具

#### 短期国债
- **国库券**：政府发行的短期债券，期限通常为3个月、6个月、1年
- 信用风险低，流动性好

#### 商业银行票据
- **银行承兑汇票**：银行承诺付款的汇票
- **商业票据**：企业发行的短期无担保本票

#### 回购协议
- **正回购**：卖出证券并约定未来回购
- **逆回购**：买入证券并约定未来卖出
- 是央行调节流动性的重要工具

#### 货币市场基金
- 投资于短期货币市场工具的基金
- 风险低，流动性好，收益相对稳定
- 适合作为现金管理工具

### 3. 货币市场主要术语
- **Shibor**：上海银行间同业拆放利率
- **Libor**：伦敦银行间同业拆放利率
- **回购利率**：资金借贷的利率
- **同业拆借**：银行间短期资金借贷

### 4. 货币市场特点
- 期限短：通常在一年以内
- 风险低：主要投资于高信用等级工具
- 流动性好：可快速变现
- 收益相对稳定

## 五、投资理财建议

### 1. 资产配置原则
- **分散投资**：不要把鸡蛋放在一个篮子里
- **风险匹配**：选择与自身风险承受能力相匹配的产品
- **长期投资**：避免短期投机，保持耐心
- **定期调整**：根据市场变化和自身情况调整配置

### 2. 不同风险等级投资者适合的产品
- **保守型**：货币基金、国债、定期存款
- **稳健型**：债券基金、混合基金
- **平衡型**：混合基金、股票型基金
- **激进型**：股票、股票型基金、衍生品

### 3. 投资注意事项
- 了解产品风险等级和投资方向
- 关注费用结构，综合考虑成本
- 关注基金管理人的专业能力
- 保持理性，避免盲目跟风
- 做好止损和止盈策略

### 4. 基金定投详解

#### 什么是基金定投？
基金定投（全称"定期定额投资"）是指投资者每隔固定时间（如每周、每月）投入固定金额到指定基金的投资方式。这种方式类似于银行的零存整取，通过长期坚持实现财富积累。

#### 基金定投的特点
- **固定金额**：每次投入的资金量固定
- **固定周期**：按约定周期（如每周、每月）持续投资
- **自动扣款**：绑定银行卡后自动完成投资
- **长期坚持**：通常需要3-5年甚至更长时间

#### 基金定投的原理
- **平均成本**：市场下跌时，同样的金额买入更多份额；市场上涨时，买入较少份额。长期下来，成本趋于平均
- **复利效应**：投资收益再投资，产生利滚利效果
- **纪律性投资**：避免情绪化操作，克服贪婪和恐惧

#### 基金定投适合人群
- 每月有固定收入的工薪阶层
- 没有时间研究市场的投资者
- 希望长期积累财富的人群
- 风险承受能力适中者

#### 基金定投操作建议
- 选择波动较大的基金（如股票型、指数型）
- 坚持长期投资，避免频繁赎回
- 设置合理的收益目标，达到后考虑止盈
- 可以设置智能定投（如估值低位多投）

### 5. 股票型基金与债券型基金的区别

| 对比项 | 股票型基金 | 债券型基金 |
|--------|-----------|-----------|
| **主要投资标的** | 股票（比例≥80%） | 债券（比例≥80%） |
| **风险等级** | 较高 | 较低 |
| **预期收益** | 较高 | 相对稳定 |
| **收益波动** | 波动较大 | 波动较小 |
| **适合投资者** | 风险承受能力强 | 风险偏好稳健 |
| **投资周期** | 适合中长期 | 适合中短期 |
| **受市场影响** | 与股市高度相关 | 与利率高度相关 |

#### 股票型基金特点
- 主要投资于股票市场
- 收益潜力大，但风险也高
- 适合追求高收益、能承受较大波动的投资者
- 受宏观经济、政策、行业发展等因素影响明显

#### 债券型基金特点
- 主要投资于债券市场
- 收益相对稳定，风险较低
- 适合追求稳健收益、风险偏好较低的投资者
- 主要受利率变动、信用风险等因素影响

#### 如何选择？
- **保守型投资者**：建议债券型基金为主，少量配置股票型基金
- **稳健型投资者**：可配置债券型基金和混合型基金
- **平衡型投资者**：股票型基金和债券型基金均衡配置
- **激进型投资者**：可以股票型基金为主，债券型基金作为配置
